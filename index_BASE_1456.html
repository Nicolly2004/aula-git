Comandos git

Configura autor do git

git config --global user.name "{Nome do usuário}"

O comando configura o nome para anexar aos commits.

git config --global user.email "{email do usuário}

O comando configura o email para anexar aos commits.

Cria um repositório git

git init

o comando, tranforma uma pasta em repositório git

Adiciona arquivos ao repositório

git add {nome do arquivo}

O comando adiciona um arquivo ou vários arquivos para o estágio de commit, para adicionar vários basta separar o nome com espaço ou usar o "." no lugar do nome.

Confirma a alteração do arquivo

git commit -m "{Mensagem de commit}"

O comando confirma as alterações com uma mensagem que deve ser descritiva e direta.

Altera a visualização do repositório

git checkout {id do comment || nome da branch}
